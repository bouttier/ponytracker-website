PonyTracker

PonyTracker is an open source and easy-to-use github-like issue tracker realised in Python with the Django web framework and released under CeCILL-C.

Warning: No registration possible yet. Accounts must be created by admin or provided by LDAP.

Main features:
  • issues classified in project
  • issues filterable by labels/milestones/author
  • markdown comments
  • global or per-project based permissions
  • per user/group/team permissions
  • emails notifications
  • very well suited for use with LDAP directory
Planed features:
  • API
  • RSS
  • Jabber notifications
  • Advanced search
  • Sorting criteria
  • authentication with social media
